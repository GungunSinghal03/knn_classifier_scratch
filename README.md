🤖 K-Nearest Neighbors (KNN) Implementation from Scratch
📌 Overview

This project implements the K-Nearest Neighbors (KNN) algorithm from scratch using Python, without using machine learning libraries like scikit-learn.
It focuses on understanding the core working of the algorithm.

🚀 What This Project Covers
KNN algorithm implementation from scratch
Euclidean distance calculation
Finding nearest neighbors
Majority voting for classification
Prediction logic
🛠️ Tech Stack
Python
NumPy
📂 Project Structure
knn_from_scratch.ipynb → Contains complete implementation
▶️ How to Run
jupyter notebook

Open:

knn_from_scratch.ipynb
📊 Key Concepts
KNN is a non-parametric, lazy learning algorithm
No explicit training phase
Predictions are based on similarity (distance)
💡 Algorithm Steps
Choose value of K
Calculate distance from input to all data points
Select K nearest neighbors
Apply majority voting
Return predicted class
🎯 Purpose

To understand the internal working of KNN by implementing it from scratch.

⚠️ Note
Basic implementation for learning purposes
No optimizations like KD-Tree used
